## 🚀 Vercel 部署完整指南

### 📋 **部署前准备工作**

#### 1. **GitHub 账号**
- 如果没有，请访问 [github.com](https://github.com) 注册账号
- 这是 Vercel 集成部署的必要条件

#### 2. **GitHub 仓库准备**
您需要将代码推送到 GitHub：

```bash
# 1. 在 GitHub 上创建新仓库（例如：smart-hr-dashboard）
# 2. 在本地初始化 Git 并推送代码：

cd e:\实习\project\AI+自动化企业赋能\智能人力资源HR\smart-hr-dashboard

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 提交代码
git commit -m "Initial commit: Smart HR Dashboard with AI Resume Parser"

# 添加远程仓库（替换为您的 GitHub 仓库地址）
git remote add origin https://github.com/您的用户名/smart-hr-dashboard.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

#### 3. **验证项目构建**
```bash
# 确保本地构建正常
npm run build
```

### 🎯 **Vercel 部署步骤**

#### **步骤1：访问 Vercel**
- 打开浏览器，访问 [vercel.com](https://vercel.com)
- 点击 "Start Deploying" 或 "Sign Up"

#### **步骤2：注册/登录**
- 选择 "Continue with GitHub" 
- 使用您的 GitHub 账号授权登录

#### **步骤3：导入项目**
- 点击 "Add New Project"
- 选择 "Import Git Repository"
- 找到您的 `smart-hr-dashboard` 仓库
- 点击 "Import"

#### **步骤4：配置部署设置**
Vercel 会自动检测到这是一个 Vite 项目，通常不需要额外配置：

**预设配置**：
- **Framework Preset**: Vite
- **Root Directory**: `/` (根目录)
- **Build Command**: `npm run build`
- **Output Directory**: `dist/public`

#### **步骤5：环境变量配置（重要）**
在部署前，需要设置环境变量：

**在 Vercel 项目设置中添加以下环境变量**：
```
VITE_ANALYTICS_ENDPOINT=https://analytics.example.com
```

**添加步骤**：
1. 在项目设置中找到 "Environment Variables"
2. 添加变量名和值
3. 选择环境：Production, Preview, Development
4. 点击 "Save"

#### **步骤6：部署**
- 点击 "Deploy" 按钮
- Vercel 开始自动构建和部署
- 通常需要 2-3 分钟

### ✅ **部署成功后**

#### **获得访问链接**
- Vercel 会提供一个默认域名：`https://your-project-name.vercel.app`
- 您可以：
  - 直接使用这个链接分享给其他人
  - 在项目设置中绑定自定义域名

#### **自动重新部署**
- 每次推送到 GitHub，Vercel 会自动重新部署
- 在 GitHub 上推送新代码，几分钟内自动生效

### 🔧 **如果遇到问题**

#### **构建失败排查**：
```bash
# 本地测试构建
npm run build

# 检查是否有 TypeScript 错误
npm run check
```

#### **常见解决方案**：
1. **环境变量缺失**：确保所有必需的环境变量都已设置
2. **构建超时**：检查 `npm run build` 是否能在本地正常工作
3. **依赖问题**：检查 `package.json` 中的依赖是否完整
